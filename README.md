PandaAI
===

Add AI player to factorio

klib
===
a library for PandaAI, build on top of factorio-stdlib, it includes:

* kcontainer
    
    A simple IOC container for factorio
    
* kagent
    
    A steer behavior agent 
    
* command

    commands for agent
    
tow
===
